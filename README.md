## 🚀 Deployment

### 🔹 Deploy on Vercel

1. Go to https://vercel.com
2. Login with GitHub
3. Click **"Add New Project"**
4. Upload or import repository
5. Ensure file name is:

```
index.html
```

6. Click **Deploy**
